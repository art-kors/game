примененные библиотеки
random - для выдачи улучшений в shmup
sys - закрытие программы
os - загрузка спрайтов/изображений
pygame - движок, на котором сделана большаяя часть программы
pygame-gui - графический интерфейс

MULTIVERSE(от англ. мультивселенная) - сборник игр, включающая в себя пока что 2 игры(реверси и shmup!).
